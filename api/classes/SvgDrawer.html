Class SvgDrawer

This is an abstract class that Shapes have to extend that to insert or draw their own

Example

editor.board.shapes.circle.insert({
 x: 100,
 y: 100,
 radius: 10,
 fill: 'red'
})

Example

editor.board.shapes.circle.draw()

Example

editor.board.shapes.circle.stopDrawing()
